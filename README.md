# Zansyz - Creative Projects Portfolio

A modern, clean portfolio website with smooth scrolling and beautiful animations.

## Features

✨ **Modern Design**
- Clean, minimal aesthetic with smooth transitions
- Responsive design that works on all devices
- Gradient backgrounds and modern typography

🎯 **Smooth Scrolling**
- CSS `scroll-behavior: smooth` for native smoothness
- JavaScript enhancements for better control
- Intersection Observer API for fade-in animations

📱 **Responsive Layout**
- Mobile-first approach
- Flexible grid system
- Touch-friendly navigation

🎨 **Interactive Elements**
- Hover effects on buttons and cards
- Navigation link animations
- Dynamic navbar background on scroll
- Staggered animations for project cards

## File Structure

```
.
├── index.html      # Main HTML structure
├── styles.css      # All styling and animations
├── script.js       # Smooth scrolling and interactivity
└── README.md       # This file
```

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser
3. Customize the content with your own projects and information

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
    /* ... more variables ... */
}
```

### Content
Update the text and structure in `index.html` to match your portfolio.

### Projects
Add more project cards by duplicating the `.project-card` element and customizing the content.
